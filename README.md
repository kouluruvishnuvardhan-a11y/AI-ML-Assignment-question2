# AI-ML-Assignment-question2
Student Performance Prediction using Linear Regression

#Overview
This project predicts student academic performance based on study-related factors using a Machine Learning model. The main goal is to analyze how variables such as study hours influence student marks and to build a predictive model using Linear Regression.

#Objective
- To analyze the relationship between study hours and student marks.
- To build a machine learning model that predicts student performance.
- To visualize the prediction using graphs.

#Dataset
The dataset contains information about students and their study hours along with corresponding marks.
Example attributes used in the dataset:
- Study Hours
- Marks Obtained
The dataset is stored in a CSV file and loaded using the pandas library.

#Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Jupyter Notebook

#Methodology
1. Import required libraries.
2. Load the dataset using pandas.
3. Separate the dataset into input features (X) and target variable (y).
4. Train the model using Linear Regression.
5. Predict student marks based on study hours.
6. Visualize the results using a regression line.

#Model
The Linear Regression model follows the equation:
Y = a + bX
Where:
- Y = Predicted Marks
- X = Study Hours
- a = Intercept
- b = Slope of the regression line

#Output
The model predicts student marks based on study hours and displays a graph showing:
- Actual student marks
- Predicted regression line

#Conclusion
The Linear Regression model successfully predicts student performance based on study hours. This type of predictive model can help educators identify patterns in student learning and support academic planning.
